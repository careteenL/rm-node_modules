![node_modules](https://careteenl.github.io/images/@careteen/rm-rf-node_modules/node_modules.jpeg)


## 快速使用

NPM
```shell
# 全局安装
npm i -g @careteen/rm-rf-node_modules

# 进入到目标目录
cd company
# 删除`company`目录下所有`node_modules`目录
careteen-rmnm
```
